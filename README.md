# Motomap

Estrutura separada em padrão de projeto front-end estático:

- `index.html` → estrutura da aplicação
- `css/styles.css` → estilos globais e componentes
- `js/app.js` → lógica do mapa, modal, popups e Firebase
- `assets/` → reservado para imagens, ícones locais e outros recursos

## Como abrir
Abra `index.html` no navegador.

## Dependências externas via CDN
- Leaflet
- Leaflet Heat
- Firebase compat
